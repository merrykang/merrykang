#### ERD 설계 → API 개발 → 클라이언트 통신 → CI/CD → 성능 개선까지 ***백엔드 개발 전반을 경험***하였습니다.
#### 최근에는 ***SpringBoot 기반 고성능 API 개발 / MLOps 엔지니어링***에 관심이 있어 지속적으로 실습 위주 학습을 진행하고 있습니다.
